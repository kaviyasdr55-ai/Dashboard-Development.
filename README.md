## Tool Used
Power BI Desktop

## Dataset Used
- Sample Financial dataset from Power BI
- Columns: Country, Sales, Profit, Year

## Dashboard Overview
- Interactive dashboard with multiple visuals
- Visuals included:
  1. Sales by Country (Column Chart)
  2. Profit by Country (Column Chart)
  3. Total Sales (Multi-row Card)
  4. Year filter (Slicer)
- Titles added for all visuals
- All charts update dynamically when a year is selected

## Actionable Insights
1. **Top-performing countries:** Shows which countries contribute most to sales.
2. **Profit Analysis:** Profit varies across countries, highlighting improvement areas.
3. **Trend Observation:** Year slicer shows sales and profit trends over time.
4. **Business Decisions:** Dashboard helps in planning promotions, targeting top markets, and monitoring performance.

## Key Features Highlighted
- Interactive slicer for filtering by year
- Automatic updates of visuals
- Clear titles for all charts
- Multi-row card for important KPIs

## Conclusion
The dashboard transforms raw financial data into clear, actionable insights.  
It helps management quickly analyze sales and profit trends, identify top markets, and make data-driven decisions.
